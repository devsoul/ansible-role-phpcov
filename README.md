PHPCOV Ansible role
=========
[![Build Status](https://travis-ci.org/devsoul/ansible-role-phpcov.svg?branch=master)](https://travis-ci.org/devsoul/ansible-role-phpcov)

If you are a PHP developer and you use phpcov.

Requirements
------------
  - `"php": "^7.1"` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phpcov

Check if it's installed:
------
    phpcov --version
